# Minimalist Dark-Themed To-Do List

Visit the live site:
A sleek, modern **dark-themed to-do list** built with HTML, CSS, and JavaScript. Easily add, edit, complete, and remove tasks with a professional minimalist design.

## Features
- Add, edit, complete, and remove tasks
- Status badges for completed/pending tasks
- Dark theme with high contrast colors
- Responsive design with hover effects

## Installation
1. Clone the repo: `git clone https://github.com/AlijaKazi/To_Do_List.git`
2. Open `index.html` in your browser
